🛠 Step-by-Step Installation Guide

1

Install MicroG

If your phone is not rooted, you must install MicroG first. This acts as a bridge to allow you to log into your Google Account safely. Download and install the APK, and make sure it runs in the background.

2

Allow Unknown Sources

Go to your phone's Settings > Security > Install unknown apps. Grant permission to your browser or file manager so you can install the downloaded APKs.

3

Install ReVanced Manager OR APK

Either install the pre-built YouTube ReVanced APK directly, or install ReVanced Manager to patch the official YouTube app on your own device.

4

Login & Enjoy

Open the app. It will connect through MicroG automatically. Sign in with your Google account and configure your settings (like SponsorBlock preferences).

❓ Frequently Asked Questions

Is YouTube ReVanced safe to use?

Yes. The patches are open-source, and building the APK yourself using the ReVanced Manager guarantees it is safe. Our pre-built APKs are compiled directly from the official source without any hidden modifications.

Do I need a rooted device?

No, root is not required. Non-rooted users simply need to install MicroG alongside ReVanced to enable Google Account sign-in capabilities.

Why is my video buffering infinitely?

This is a known issue caused by YouTube's backend changes. Ensure you are using the latest version of ReVanced and MicroG. Sometimes enabling the "Spoof App Version" toggle in ReVanced settings fixes this instantly.

What is the difference between ReVanced and Extended (RVX)?

ReVanced is the core official project. RVX (Extended) is a fork created by independent developers that includes extra community-requested features and patches that the official team rejected or hasn't implemented yet.
